## prerequisites

* PHP 8.3
* Composer

## installation

`composer g require hexlet/code`

## usage

```sh
Generate diff

Usage:
  gendiff (-h|--help)
  gendiff [--format <fmt>] <firstFile> <secondFile>

Options:
  -h --help                     Show this screen
  --format <fmt>                Report format [default: pretty]
```
